## 命名規約(Python)
- pythonの命名規約はPEP8(以下の通り)に従う
| 項目       | 命名規則         |
| ---------- | ---------------- |
| ファイル名 | lower_snake_case |
| クラス名   | CamelCase        |
| メソッド名 | lower_snake_case |
| 定数名     | UPPER_SNAKE_CASE |
| 変数名     | lower_snake_case |
| 引数名     | lower_snake_case |
